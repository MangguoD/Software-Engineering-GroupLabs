# 家教服务系统后端 (Flask)

本项目为家教服务系统的后端部分，使用 Python Flask 编写，提供用户注册登录、家教信息发布、学生需求发布、信息查询、匹配推荐、评价系统与简单问答功能的 API 接口。

---

## 📦 安装依赖

确保你的系统已安装 Python 3.7 或以上版本。

进入 backend 目录，安装依赖：

```bash
pip install -r requirements.txt
```

如果使用 MySQL 数据库，请确保安装了 pymysql：
```bash
pip install pymysql
```
